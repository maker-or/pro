## 🧱 Technical Requirements

### Runtime & Package Management

- The application must run on **Bun only**
- Use **bun** as the package manager
- Always refer to `package.json` before running commands
- Never use npm, pnpm, or yarn
- Always run bun
- When ever a particular file is edited , always run `bun run lint` and run `bun run format`


---

### Type System & Core Architecture

- Must be built using **Effect (TypeScript library)**
- Type safety is **non-negotiable**
- Avoid unsafe patterns
- Prefer explicit types over inference in critical paths
- Favor composable, functional architecture

---

### Frontend Standards

- Always apply strong frontend design principles
- Use **shadcn** for UI components
- Always use your **frontend-design** skill in agents while designing the frontend UI
